# Методология исследования (шаблон для магистерской)

## 1. Цель

Оценить эффективность визуальных трансформеров в задаче обнаружения БПЛА в сложных условиях наблюдения:

- плохая погода (дождь, снег, туман)
- низкое качество изображения и шум
- быстрые маневры и резкие изменения масштаба
- малый размер цели в кадре

## 2. Гипотезы

1. ViT/DETR-подобные модели дают более устойчивый `mAP50` в сложной погоде по сравнению с CNN-бейзлайнами.
2. При малых объектах рост точности ViT достигается ценой увеличения latency.
3. Предобученные трансформерные детекторы лучше переносят доменный сдвиг между условиями съемки.

## 3. Дизайн эксперимента

- Единый формат данных: COCO
- Сплиты: `train/val/test` с контролем по условиям (`weather`, `quality`, `maneuver`)
- Модели:
  - `detr_resnet50` (базовый гибридный детектор)
  - `yolos_tiny` (ViT-подход)
  - собственные варианты через `register_model`
- Метрики:
  - `mAP@[.50:.95]`, `mAP50`, `mAP75`, `mAR`
  - latency (ms) и FPS

## 4. Статистическая проверка

- Сравнение моделей по bootstrap CI для `mAP50`
- Для попарных сравнений по условиям использовать непараметрический тест (например Wilcoxon), если распределения не нормальны

## 5. Анализ ошибок

- Разбивка ложных срабатываний и пропусков по:
  - погодным условиям
  - качеству видео
  - типу маневра
  - размеру объекта (small/medium/large)

## 6. Формат выводов

1. Общий лидер по точности
2. Лидер по эффективности (`mAP50 / latency`)
3. Условия, где ViT дает максимальный выигрыш
4. Условия, где CNN/гибрид оказывается практичнее
5. Рекомендация для deployment (edge vs server)
