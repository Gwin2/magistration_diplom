# Results Section Template

## Main Benchmark Table

Use `reports/summary.tex` generated by:

```bash
uav-vit summarize --runs-dir runs --output-dir reports
```

## Narrative Template

1. Best `mAP50` achieved by `...` with value `...`.
2. Fastest model by latency is `...` with `... ms` per batch.
3. Best efficiency (`mAP50 / ms`) is `...`.
4. In difficult weather conditions (`...`), model `...` improved by `...%` over baseline.
5. Failure patterns indicate issues on `...` (e.g., blur + small target).

## Figure Checklist

- Learning curves: train loss and val mAP by epoch
- Condition-wise mAP (weather / quality / maneuver)
- Precision-recall curves
- Example detections (TP/FP/FN cases)
